Privacy Policy for PasswordManager

PasswordManager securely stores users’ website usernames, passwords, comments, and record date/time locally in the user’s Chrome browser storage (Chrome Storage API). No data is collected, transmitted, or accessible by the extension developer or any third party. All information remains on the user’s device and can only be decrypted with the user’s master password.

Data Collection and Storage

The extension does not collect any data outside the user's device. The only data stored is the information users explicitly save (usernames, passwords, comments, date/time) which is encrypted.

Permissions

  • Storage: To store encrypted account and password data locally.
  • ActiveTab: To detect the current tab URL and show the "Save To Password Manager" prompt.
  • Scripting: To inject content scripts to detect login fields on web pages.
  • Host permissions (<all_urls>): To access login forms on any website to offer password saving.

Third-Party Services

No third-party services are used. All code runs locally within the extension package.

Data Security

All stored data is encrypted using the user's master password. No password or sensitive data leaves the user’s device.
